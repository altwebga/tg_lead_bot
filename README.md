# Telegram Order Bot

Telegram-бот для приема заявок на разработку, доработку сайтов и SEO/продвижение. Бот проводит пользователя по короткому wizard-сценарию: выбор услуги, описание задачи, бюджет и контакт. После заполнения заявка отправляется администратору в Telegram.

## Возможности

- Пошаговый сценарий оформления заявки
- Inline-кнопки для выбора услуги и бюджета
- Отправка заявки в админский чат
- Валидация обязательных переменных окружения при старте
- Защита HTML-разметки от пользовательского ввода
- Команды `/start` для новой заявки и `/cancel` для отмены

## Стек

- Node.js
- TypeScript
- Telegraf
- dotenv

## Установка

```bash
npm install
```

## Команды

Запуск в режиме разработки:

```bash
npm run dev
```

Сборка TypeScript:

```bash
npm run build
```

Запуск собранной версии:

```bash
npm start
```

## Структура проекта

```text
src/
  bot.ts            # Точка входа, подключение middleware и запуск бота
  config.ts         # Проверка и экспорт переменных окружения
  keyboards.ts      # Inline-клавиатуры и callback-значения
  types.ts          # Типы контекста, сессии и заявки
  scenes/
    order.ts        # Wizard-сценарий оформления заявки
```

## Сценарий заявки

1. Пользователь отправляет `/start`
2. Выбирает услугу
3. Описывает задачу
4. Выбирает бюджет
5. Оставляет контакт
6. Бот отправляет заявку администратору
